## 三、农业碳排放核算框架

### 3.1 核算范围

```
农业碳排放 = 农业投入品隐含碳 + 稻田甲烷排放 + 农业能源消耗排放 - 土壤碳汇
```

### 3.2 核算公式（已验证可用）

#### 总碳排放公式
$$E_t = E_{input} + E_{CH_4} + E_{energy} - S_{soil}$$

#### 农业投入品隐含碳
$$E_{input} = \sum_{m} (X_{mt} \times \delta_m)$$

#### 稻田甲烷排放
$$E_{CH_4} = A_t \times (E_{FRGS} \times T_s + E_{FnonRGS} \times T_{non}) \times GWP_{CH_4}$$

#### 农业能源消耗排放
$$E_{energy} = \sum_{k} (F_{kt} \times EF_k)$$

## 四、关键变量与符号说明

| 符号 | 含义 | 单位 | 数据来源 |
|------|------|------|----------|
| $E_t$ | 第t年农业碳排放净量 | 万吨CO2e | 计算得出 |
| $E_{input}$ | 农业投入品隐含碳排放 | 万吨CO2e | 统计年鉴 |
| $E_{CH_4}$ | 稻田甲烷折算CO2当量 | 万吨CO2e | 计算得出 |
| $E_{energy}$ | 农业能源消耗排放 | 万吨CO2e | 能源统计 |
| $S_{soil}$ | 土壤碳汇量 | 万吨CO2e | 土壤检测 |
| $X_{mt}$ | 第m类农业投入品使用量 | 吨 | 统计年鉴 |
| $\delta_m$ | 碳排放系数 | kg CO2e/kg | IPCC清单 |
| $A_t$ | 水稻种植面积 | 万亩 | 统计年鉴 |
| $E_{FRGS}$ | 水稻生长季甲烷排放因子 | g·m⁻² | 本地化研究 |
| $E_{FnonRGS}$ | 非水稻生长季排放因子 | g·m⁻² | 本地化研究 |
| $GWP_{CH_4}$ | 甲烷全球增温潜势 | 28 | IPCC |
| $F_{kt}$ | 第k类农业能源消费量 | 吨 | 能源统计 |
| $EF_k$ | 能源排放系数 | kg CO2e/kg | IPCC清单 |
